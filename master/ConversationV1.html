9/20 update made changes to response format

  • context.system.dialog_stack changed from an array of strings to an array of objects

Old:

    "system": {
      "dialog_stack": [
        "root"
      ],

New:

 "context": {
    "system": {
      "dialog_stack": [
        {
          "dialog_node": "root"
        }
